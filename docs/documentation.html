LawFIRM Template Documentation

Complete guide to using and customizing the LawFIRM HTML template.

Getting Started

Follow these steps to get your law firm website up and running:

  1. Extract the downloaded ZIP file
  2. Open the template folder in your code editor
  3. Use a local web server (Live Server, XAMPP, etc.)
  4. Navigate to pages/index.html in your browser
Important: This template uses the Fetch API to load partials. You must use a web server - opening files directly will not work properly.

Customization

Colors

Edit CSS variables in assets/css/style.css:

:root {
  --primary-navy: #0a1c2e;
  --primary-gold: #c9a142;
  --secondary-gold: #d4af37;
}

Fonts

Change fonts by updating:

  • Google Fonts link in HTML files
  • CSS variables in style.css

Logo

Replace text logo in partials/header.html with your image logo.

Deployment Guide

GitHub Pages:

  1. Create a new repository on GitHub
  2. Upload all files to the repository
  3. Go to Settings → Pages
  4. Select branch and folder
  5. Your site will be live!

Other Hosting:

Upload files via FTP to any web hosting service. Ensure all paths are correct.
